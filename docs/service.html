AI Commit Messages

noidea uses advanced AI to analyze your staged changes and generate meaningful commit messages that accurately describe what you've done. No more struggling with what to write!

The AI model is trained on high-quality commit messages and can identify patterns in your code changes to suggest appropriate descriptions. You can get suggestions on demand with the command noidea suggest or enable automatic suggestions with git config noidea.suggest true.

Benefits include:

  • More professional Git history
  • Time saved on writing messages
  • Consistent message format
  • Better collaboration with team members

Sassy Moai Feedback

After each commit, the Moai will judge your work with witty, sassy comments that make Git more entertaining. The Moai analyzes your commit message quality, code changes, and commit patterns.

The feedback ranges from encouraging to sarcastic, depending on the personality you've chosen and the quality of your commit. It's a fun way to add some personality to your development workflow.

Sample Moai reactions include:

  • "Ah, another 'final fix'. I've heard that one before."
  • "This commit is clean. Almost suspiciously clean..."
  • "You've entered the 2AM hotfix arc. A legendary time."
  • "Did you just refactor the entire codebase? Ambitious!"

Git History Analysis

noidea can analyze your Git history to provide insights about your development patterns. Use the noidea summary command to get a detailed report about your recent activity.

The analysis includes information about your most active times, common commit types, lines of code changed, and more. This helps you understand your workflow better and identify areas for improvement.

Key metrics include:

  • Commit frequency by day and hour
  • File types most commonly modified
  • Average commit size
  • Common commit patterns
  • Project activity over time
